# Trello to WhatsApp BridgeUma extensăo para Chrome/Brave projetada para automatizar o reporte de progresso de tarefas do Trello diretamente para o WhatsApp Desktop/Web. Ideal para desenvolvedores e gestores que utilizam o Trello como ferramenta de Kanban e o WhatsApp para comunicaçăo de status em tempo real.## Funcionalidades- **Check-in Instantâneo:** Botăo "Iniciar" que gera uma mensagem formatada com a hora exata do início da tarefa.- **Check-out (Entrega):** Botăo "Entregar" para reportar a conclusăo com link direto para o card para revisăo (QA).- **Extraçăo de ID Dinâmica:** Captura automaticamente o número do chamado (ex: `#DEV-123`) de campos personalizados do Trello.- **Protocolo Híbrido:** Tenta abrir o **WhatsApp Desktop** instalado no sistema; caso falhe, redireciona para o WhatsApp Web.- **Modo Debug/Teste:** Botăo dedicado para validar a captura de dados sem sujar os logs oficiais.## Tecnologias- **JavaScript (ES6+)**- **Chrome Extension API (Manifest V3)**- **Regex** para extraçăo de dados dinâmicos.## Como Instalar (Modo Desenvolvedor)1. Faça o download ou clone este repositório.2. Abra o Google Chrome ou Brave e acesse `chrome://extensions`.3. Ative o **Modo do Desenvolvedor** no canto superior direito.4. Clique em **Carregar sem compactaçăo** e selecione a pasta do projeto.5. Abra qualquer card no Trello e os botőes aparecerăo automaticamente na barra lateral.## Ambiente de TesteA extensăo possui um botăo de **Teste de Integraçăo**. Ao clicar nele:1. O prefixo da mensagem será alterado para `?? [TESTE]`.2. Os dados capturados (Título, ID, URL) serăo exibidos no Console do Navegador (`F12`).3. Isso permite validar se a extraçăo de campos customizados está funcionando antes do envio real.## Exemplo de Mensagem Gerada> *[INICIANDO]*> **Hora:** 08:30> **Tarefa:** Implementaçăo do Result Pattern no Backend> **ID:** #DEV-2025596> **Link:** https://trello.com/c/abc123xyz## LicençaEste projeto é open-source.
